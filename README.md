# Demo-LTQL
Các chức năng:
1. Form giới thiệu và cấu hình, ghi log, tool mật khẩu
2. Đăng nhập, đăng xuất, quên mật khẩu, đổi mật khẩu.
3. Phân quyền cơ bản, phân quyền nâng cao
4. MDI - Docking - Form main, Form child
5. Form quản lý (thêm xóa sửa xem tìm kiếm)
6. Form giao diện trực quan
7. Nhập - Xuất (Import - Export)
8. Tìm kiếm cơ bản, tìm kiếm nâng cao
9. Report (danh sách, master-details, tham số, biểu đồ)
10. Sao lưu, phục hồi, Lưu trữ dữ liệu cuốn chiếu, Setup, Update phần mềm

Phi chức năng:
1. Tài liệu phân tích thiết kế
2. Yêu cầu thiết kế dữ liệu, xử lý, giao diện
3. Bảo mật ứng dụng
